# Rules

- Complete only the requested task
- Never ask for confirmation
- Never wait for user input
- Never inspect unrelated files
- Open only files required for the task
- Minimize token usage
- Minimize repository exploration
- Avoid broad searches
- Avoid architecture analysis unless explicitly requested
- Do not read the entire repository
- Do not run builds unless explicitly requested
- Do not run npm install unless necessary
- Do not run dev servers
- Never leave background processes running
- Avoid unnecessary retries
- Prefer minimal code changes
- Prefer localized modifications
- Finish the task completely
